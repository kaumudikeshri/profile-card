# Profile Card

A simple profile card built using HTML and CSS.

## Features

- Centered profile card layout
- Profile image section
- Name and short description
- Follow button
- Button hover effect
- Border, border-radius, and box-shadow styling

## Technologies Used

- HTML5
- CSS3

## What I Practiced

- Creating a card layout using `<div>`
- Adding and styling an image using `<img>`
- Using Flexbox to align content
- Styling text, buttons, borders, and shadows
- Adding hover effects with CSS
